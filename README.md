# Film Review Backend

এই প্রজেক্টটি **Film Review** ওয়েব অ্যাপের ব্যাকএন্ড সার্ভার। এটি **Node.js**, **Express.js**, **MongoDB**, এবং **Firebase Authentication** ব্যবহার করে তৈরি করা হয়েছে। সার্ভারটি মুভি, ওয়াচলিস্ট এবং ইউজার-স্পেসিফিক কালেকশন পরিচালনার জন্য API প্রদান করে।

---

## ফিচারসমূহ

- **মুভি ম্যানেজমেন্ট**

  - সব মুভি দেখুন বা শিরোনাম দিয়ে সার্চ করুন।
  - Genre অনুযায়ী মুভি ফিল্টার করুন।
  - নতুন মুভি যোগ, আপডেট বা ডিলিট করতে পারবেন।
  - মুভির বিস্তারিত তথ্য দেখুন (ইউজার অথেন্টিকেশন প্রয়োজন)।

- **ইউজারের ওয়াচলিস্ট**

  - মুভি ওয়াচলিস্টে যোগ করুন।
  - নিজের ওয়াচলিস্ট দেখুন (অথেন্টিকেশন প্রয়োজন)।
  - ওয়াচলিস্ট থেকে মুভি সরান।

- **ইউজারের কালেকশন**

  - নির্দিষ্ট ইউজার তৈরি করা মুভিগুলো দেখুন (অথেন্টিকেশন প্রয়োজন)।

- **অথেন্টিকেশন**

  - Firebase Authentication ব্যবহার করে সিকিউর রুট প্রটেকশন।

- **ডেটাবেস**
  - **MongoDB** ব্যবহার করা হয়েছে মুভি এবং ওয়াচলিস্ট ডেটা সংরক্ষণের জন্য।

---

## ব্যবহার করা টেকনোলজি

- **Node.js** – জাভাস্ক্রিপ্ট রানটাইম।
- **Express.js** – ব্যাকএন্ড ফ্রেমওয়ার্ক।
- **MongoDB** – নোSQL ডেটাবেস।
- **Firebase Admin SDK** – ইউজার অথেন্টিকেশন এবং টোকেন ভেরিফিকেশন।
- **dotenv** – এনভায়রনমেন্ট ভ্যারিয়েবল ম্যানেজমেন্ট।
- **cors** – ক্রস-অরিজিন রিকোয়েস্ট সক্ষম করা।

## API Endpoints

| মেথড   | এন্ডপয়েন্ট                     | বর্ণনা                                      | অথেন্টিকেশন প্রয়োজন |
| ------ | ------------------------------ | ------------------------------------------- | -------------------- |
| GET    | `/movies`                      | সব মুভি দেখুন বা জেনার অনুসারে ফিল্টার করুন | ❌                   |
| GET    | `/search?search=<query>`       | শিরোনাম দিয়ে মুভি সার্চ করুন                | ❌                   |
| GET    | `/movie-details/:id`           | ID অনুযায়ী মুভি বিস্তারিত দেখুন             | ✅                   |
| POST   | `/movies`                      | নতুন মুভি যোগ করুন                          | ✅                   |
| PUT    | `/movies/:id`                  | মুভি তথ্য আপডেট করুন                        | ✅                   |
| DELETE | `/movie/:id`                   | ID অনুযায়ী মুভি ডিলিট করুন                  | ❌                   |
| GET    | `/my-collection?email=<email>` | ইউজার তৈরি করা মুভি দেখুন                   | ✅                   |
| POST   | `/watchlist/:id`               | মুভি ওয়াচলিস্টে যোগ করুন                    | ❌                   |
| GET    | `/my-watchlist?email=<email>`  | ইউজারের ওয়াচলিস্ট দেখুন                     | ✅                   |
| DELETE | `/my-watchlist/:id`            | ওয়াচলিস্ট থেকে মুভি সরান                    | ❌                   |

## Environment Variables

প্রজেক্টের মূল ফোল্ডারে `.env` ফাইল তৈরি করুন এবং নিচের ভ্যারিয়েবলগুলো যোগ করুন:

```env
PORT=3000
DB_USERNAME=<আপনার_mongodb_username>
DB_PASSWORD=<আপনার_mongodb_password>
FIREBASE_SERVICE_KEY=<আপনার_base64_encoded_firebase_service_key>
```
